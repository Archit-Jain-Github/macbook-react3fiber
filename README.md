# 💻 MacBook 3D – React Three Fiber Project

A minimal 3D showcase of a MacBook using **React**, **React Three Fiber**, and **Tailwind CSS**. Includes scroll animations, camera controls, and texture mapping.

🔗 [Live Demo](https://macbook-react3fiber.pages.dev)
📦 [GitHub Repo](https://github.com/Archit-Jain-Github/macbook-react3fiber)

---

## 🚀 Quick Start

```bash
git clone https://github.com/Archit-Jain-Github/macbook-react3fiber.git
cd macbook-react3fiber
npm install
npm run dev
```

Open [localhost:5173](http://localhost:5173) to view it in the browser.

---

## 🔧 Tech Stack

- **Vite** – Build tool
- **React** – UI library
- **React Three Fiber** – Three.js in React
- **Drei** – Helpers for R3F
- **Tailwind CSS** – Utility-first CSS
